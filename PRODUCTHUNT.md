# ProductHunt Launch — Solar System Agents

## Listing Details

**Name:** Solar System Agents

**Tagline (60 chars max):**
Visualize your AI agents as an interactive solar system

**Description:**
Solar System Agents is an open-source, zero-dependency dashboard that turns your AI agent fleet into a real-time solar system.

Each agent becomes a planet — orbiting, glowing, and pulsing based on live status. The sun is your central gateway. Moons are sub-agents. Rings indicate scheduled tasks. Click any planet to see detailed metrics.

It works with any multi-agent framework: CrewAI, LangChain, AutoGPT, OpenAI Assistants, Semantic Kernel, or custom setups.

One HTML file. No build step. No npm install. Just edit a JavaScript array with your agents and open in a browser.

**Key Features:**
- Interactive solar system rendered in Canvas2D at 60fps
- Each planet = one AI agent with real-time status
- Moons for sub-agents, rings for scheduled tasks
- Click-to-select with detailed stats panel
- Zoom, pan, and smooth orbital animations
- Animated starfield background
- Zero dependencies — single HTML file
- Framework agnostic
- MIT licensed

## First Comment (post immediately after launch):

"Hey Product Hunt! I built Solar System Agents because I run 18 AI agents for my business and had no decent way to monitor them all at once.

Existing tools are either too technical (log files), too generic (Grafana), or locked to one framework (LangSmith).

I wanted something that:
1. Looks incredible (because if I'm staring at a dashboard all day, it should look good)
2. Works with ANY agent framework
3. Has zero setup friction

The result: one HTML file where each agent is a planet in a solar system. Edit a JSON array, open in browser, done.

It's fully open source (MIT) and I'd love contributions — especially adapters for CrewAI, LangChain, and AutoGPT.

What would you build with it?"
